# SalonAI

AI-powered scheduling and recommendation system for hair salons.

## Overview

SalonAI streamlines appointment scheduling, automates price quoting, and offers personalized hairstyle and color recommendations based on client history.

## Tech Stack

- **Frontend:** Next.js (React + TypeScript)
- **Backend:** Flask (Python)
- **Database:** PostgreSQL
- **AI:** Custom recommendation engine
